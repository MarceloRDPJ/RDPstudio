# Igreja Casa Hub

> **Status:** Em operação | **Tech:** Cloudflare Pages, Supabase, JavaScript Vanilla

Plataforma de gestão para a Igreja Casa com painel administrativo, CRM pastoral, formulários públicos, analytics, escala, DOMUM, Wi-Fi e automações de comunicação.

O objetivo desta página no portfólio é explicar a solução, a arquitetura e o valor operacional entregue. A demonstração pública não está disponível.
